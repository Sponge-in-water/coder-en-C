les tableaux

un tableau en programmation est une variable qui contient un nombre définis d'éléments de même type ce nombre s'appelle la taille du tableau, ils sont utile pour stocker un bon nombre de valeur de même type sans avoir à crée une variable pour chaque valeur

il existe plusieur manière de l'initialiser :


        #include <stdio.h>
        
        int main(){
            
            int tab1[5] = {1,2,3,4,5}; -> ici on explicite la taille entre crochet et on lui les donne les valeurs

            int tab2[] = {1,2,3,4,5}; -> ici on ne dit pas la taille, le code le déduit

            int tab3[3];  -> ici on déclare le tableau puis on lui donne les valeurs
            tab3[0] = 1; 
            tab3[1] = 2;  après la déclaration quand on lui donne des valeurs on lui dis la position du nombre entre crochet
            tab3[2] = 3;

            return 0;
        }

comme vu dans l'exmple les éléments du tableau, ont chacun une postion qui varie entre 0 et taille-1, si ton tableau est de taille 5, la postion des éléments varie entre 0 et 4, ce qui est utile a savoir quand on intialise les valeurs après déclaration

Schéma pour comprendre

int tab1[3] = 10 25 31 -> taille = 3
postion : 0 position : 1 postion : 2

Récapitulatif

initialisation specificité
int tab[2]={1,2}; --
int tab[]={1,2}; --
int tab[2]; tab[0]=1; tab[1]=2; dans la déclaration le crochet définit la taille et dans l'initalisation des valeurs les crochets dis la position de la valeur dans le tableau

Les conditions

on sait faire certaine opération, mais on aimerais aussi pouvoir avoir des conditions par exemple : si 1+3 > 3 alors on écrit que 1+3 est plus grand que 3.

pour cela on va utiliser des fonctions if ,else if et else


        #include <stdio.h>
        
        int main(){
            
            int x = 3;

            int y = 5;

            if(x+y > 2){
                printf(x plus y est plus grand que 2);
            }

            else if(x+y < 2){
                printf(x plus y est plus petit que 2);
            }

            else{
                printf(x plus y est ni plus grand ni plus petit que 2);
            }

            return 0;
        }

ici la condition if te dis "si x+2 est plus grand que 2 alors fais la commande qui se trouve dans les accolade {}, si ce n'est pas le cas continue le code normalement"

la condition else if est juste la suite du if on peut le traduire par "autrement x+2 est plus petit que 2 alors fais la commande qui se trouve dans les accolade {}, si ce n'est pas le cas continue le code normalement"

et si aucune de ces deux condition est rempli alors on fait ce qui se retrouve dans les accolades

Note : les symboles mathématiques pour comparer sont :

  • < : est inférieur à
  • > : est supérieur à
  • <= : est inférieur ou égal à
  • >= : est supérieur ou égale à
  • == : est égale à   ici le égale est différent de = , le double pour comparer, le simple pour affecter
  • != : est différent de
