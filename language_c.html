        #include <stdio.h>
        
        int main(){

            return 0;
        }

variable et type de valeurs

une variable est une valeur (un nombre, un nombre a virgule etc ...) que tu définis, et tu le fais en fonction de type qui va définir si c'est un nombre entier (1 , 2 , 3 , 4 ...), un nombre a virgule (1.2 , 2.3 ...) ou encore un caractère unique.
exemple comme suit :


        #include <stdio.h>
        
        int main(){

            int x = 5;      -> ici x vaut 5 et "int" définis le fais que x est un entier 

            float y = 4.5;   -> ici y vaut 4.5 et "float" définis le fais que y est un nombre a virgule 

            char x = 'A'; -> ici x vaut A et "char" définis le fais que ce soit un caractère

            return 0;
        }

A savoir qu'une variable peut être modifié au cours d'un code et peux ne pas avoir de valeur au départ

exemple


        #include <stdio.h>
        
        int main(){

            int x;

            x = 6;  note : on a pas besoin d'écrire le int car la variable a été initialiser juste en haut

            x = 10;

            return 0;
        }

        a la fin du code x vaut 10 et non 6

Récapitulatif

type des variables fonctions
int nombre entier
float nombre a virgule
char Un seul caractère

calcule simple

ces variables peuvent être utile pour calculer : additionner, soustraire, multiplier, diviser la division est entière donc il n'y aura pas de virgule peu importe les valeurs calculer

exemple comme suit :


        #include <stdio.h>
        
        int main(){

            int x = 5;
            int y = 2;

            int addition = x+y;  -> ici la variable addition a pour valeur 7

            int soustraction = x-y; -> ici la variable soustraction a pour valeur 3

            int multiplication = x*y; -> ici la variable multiplication a pour valeur 10

            int division = x/y; -> ici la variable division a pour valeur 2
        }

Récapitulatif

symbole fonctions
+ somme
- soustraction
* multiplication
/ division entière

affichage et interaction

on peut calculer mais il faut pouvoir afficher ces résultats, ainsi on introduit les fonctions printf et scanf, printf c'est pour l'afficher, et scanf pour demander des valeurs a l'utilisateur sans accéder au code
exemple : 


        #include <stdio.h>
        
        int main(){

            int x;

            scanf("%d", &x); -> ici la fonction va attendre une entrée de l'utilisateur dans le terminale pour affecter la valeur entrée a la variable x

            int y = 2;

            int resultats = x+y;  -> ici la variable résultats a pour valeur (x+2)

            printf("le résultat est %d", resultats); -> ici la fonction va afficher la valeur de la variable "resultats" dans le terminale avec du texte
        }

    note : le %d correspond au type de la variable, si les valeurs auraient été des nombres a virgule (aussi dit des floats) on aurait mis %f

la fonction printf est quasiment vitale pour savoir si ce que tu fais fonctionne comme voulu utilise le souvent

Récapitulatif

commande fonction
printf affichage de texte ou valeur
scanf acquisiton de valeurs entré par l'utilisateur

le terminale

le terminale te permet d'entrer des commandes, pour compiler ton programme (compiler c'est en gros dire a ton ordinateur de traduire le code dans son language), pour éxecuter ton programme (le lancer), crée des fichiers, des dossiers, en supprimer etc...

pour y accéder tu as deux moyens soit tu ouvres l'application qui est déjà télécharger dans ton ordinateur, soit tu l'ouvres depuis Vscode, regarde l`images pour savoir comment faire depuis VScode(regarde le curseur en haut a droite)

maintenant que c'est ouvert tu vas devoir compiler ton programme puis l'éxecuter tu devras tapper 2 commandes 


        gcc -Wall [nom.c]  -> commande pour compiler, [nom.c] correspond au nom du fichier 

        et

        ./a.out ->commande pour éxecuter

./a.out éxecute le programme
